Software

The following Open Source Software forms a critical part of the spatial data infrastructure within Saint Lucia, and users are encouraged to utilise these products and services as extensively as is practical in line with Open Data Standards.

QGIS

Open Source GIS

QGIS Desktop and QGIS Server are fully featured Geographic Information Systems and are the de facto leader in Open Source Desktop GIS.

Input

Mobile Field Survey Solution

The Input App leverages the QGIS ecosystem and the Mergin cloud platform for simple and effective field data capture.

PostGIS

Spatial and Geographic objects for PostgreSQL

PostGIS is a spatial database extender for PostgreSQL object-relational database. It adds support for geographic objects allowing location queries to be run in SQL.

Geoservices

By the Government of Saint Lucia

The Geoservices Platform provides a number of services, content, components, and data for geospatial solution support in and around Saint Lucia.

This site is a component of the OpenSource GIS Stack, a full featured complete stack with end-to-end workflows for many common GIS Infrastructure components.

Built and deployed by Kartoza as a component of the 2021 Open Source GIS Training programme, delivered in line with the objectives of the Disaster Vulnerability Reduction Project (DVRP).

The OpenSource GIS Stack is a composition of various Free and Open Source software, platforms, and components. The source code is available on GitHub, and community contributions are welcome.
